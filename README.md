# 52°North uDig WPS Plugin

uDig is a Desktop GIS written in Java and based on the eclipse platform.
It is a free and open source software project driven by Refractions
Research. This application adds the ability to the uDig system to interact with
processes served by Web Processing Services implementing version
1.0.0 of the OGC specification.

The plugin allows to select WPS processes from a list. 
The selected process can be executed using layers from uDig and input provided by the user. 
The result can be displayed in the map view as a new layer.
